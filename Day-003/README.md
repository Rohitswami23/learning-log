# Day 003

**Date:** 19 July 2026

## Networking

### Topic
Security Fundamentals

### What I Learned

- I studied the CIA Triad (Confidentiality, Integrity, and Availability) and why it is the foundation of information security.
- I learned the difference between vulnerabilities, threats, exploits, and mitigation techniques.
- I explored common attacks, including DoS and TCP SYN Flood attacks.
- I studied spoofing attacks such as DHCP spoofing and DNS reflection/amplification attacks.
- I learned how Man-in-the-Middle (MITM) attacks work using techniques like ARP spoofing.
- I studied reconnaissance as the first phase of an attack.
- I learned about different types of malware, including viruses and worms.
- I explored social engineering attacks such as phishing, vishing, and watering hole attacks.
- I studied password attacks, including brute-force and dictionary attacks.
- I learned the basics of AAA (Authentication, Authorization, and Accounting) and Multi-Factor Authentication (MFA).

## Linux

### Topic
Linux Kernel Fundamentals

### What I Learned

- I studied the Linux architecture and understood how the kernel acts as the bridge between applications and hardware.
- I learned the difference between User Mode and Kernel Mode.
- I understood how system calls allow applications to request services from the kernel.
- I learned how to check and update the Linux kernel.
- I explored important files inside the `/boot` directory, including `vmlinuz`, `initramfs`, `System.map`, and `config`.
- I studied kernel modules, how they extend kernel functionality, and practiced commands such as:
  - `uname -r`
  - `lsmod`
  - `modprobe`
